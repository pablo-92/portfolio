    .encriptador

    Lenguajes

    El encriptador es un programa sencillo que permite al usuario escribir el mensaje que quiera que sea encriptado (siempre y cuando el texto no contenga mayusculas, ni tildes).


    El botón "encriptar" toma al texto y, con el uso de "llaves", codifica el mismo para que no pueda ser interpretado por terceros. Dando como resultado, un texto que no tiene ningun sentido para todo aquel que desconozca el método de decodificación.


    Por su parte, el receptor puede colocar el texto encriptado y decodificarlo para obtener el mensaje original. Utilizando el proceso inverso al de codificación.
